# Tag와 Element의 차이

많은 사람들이 태그와 요소를 같은 의미로 사용하지만, 사실 태그와 요소는 의미가 다릅니다.

태그는 <,> 기호로 표현된 규칙이며, 요소는 내용을 포함한 태그 전체를 뜻 하죠.

```
그러므로 <h1></h1> 은 태그(Tag),

<h1>Hello, HTML</h1> 은 요소(Element)라고 합니다.
```
